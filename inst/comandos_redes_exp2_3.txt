-> configurar VLANS

configure terminal
vlan 40
end
show vlan id 40

configure terminal
interface fastethernet 0/x (x: numero da porta)
switchport mode access
switchport access vlan 40
end
show running - config interface fastethernet 0/x
show interfaces fastethernet 0/x switchport
